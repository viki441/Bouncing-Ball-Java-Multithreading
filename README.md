# Bouncing Balls – JavaFX Simulation

A **multithreaded, interactive JavaFX simulation** of bouncing balls with obstacles.  
Demonstrates animation, collision detection, and responsive UI controls.

## Features

- Real-time ball animation using `AnimationTimer`
- Collision detection:
  - Ball-to-wall
  - Ball-to-ball
  - Ball-to-obstacle
- Interactive controls:
  - Spawn balls by clicking the playground
  - Adjust ball speed via a slider
  - Change ball color with a ColorPicker
  - Pause/Resume simulation
- Obstacle support with rectangular shapes
- Clean UI styled with CSS
